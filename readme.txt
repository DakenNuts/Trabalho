===========
ORGANIZAÇÃO
===========
• Pasta 'images': É a pasta responsável por conter as imagens utilizadas no código
• Pasta 'images_test': Imagens utilizadas na apresentação do grupo para a sala, anteriormente utilizadas como teste
• Pasta 'classify_images': Imagens separadas com 10 fotos para cada classe, interessante para maior base de dados para análise de gráfico.

==================
EXECUÇÃO do CÓDIGO
==================
1. Executar comandos: pip install seaborn
					  pip install seaborn pandas
					  pip install opencv-python
					  pip install matplotlib       (caso ainda não instalados)
					  pip install numpy
					  pip install tensorflow
					  pip install scikit-learn
					
2. Executar comando: python projeto.py

pressione 'q' para passar por cada janela exibida na tela.
